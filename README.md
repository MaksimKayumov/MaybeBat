# MaybeBat
My esolang in batch
# Get Started
## Compiler Usage
To view version:

```maybebat --ver```

To Compile Code:

```maybebat --compile path/to/file(without extension)```

To Compile and Run:

```maybebat --run path/to/file(without extension)```
NOTE: file extension should be .maybebat
